## Rule

Gridwork installs factory definitions only. It does not generate frontend, backend, database or Docker application code.

After installation, stack skills may be used by authorized agents inside confirmed paths and approved workflows. `repository-bootstrap` may create approved framework scaffolds and delivery foundations, but never business behavior. Skills never expand permissions.
