# Etiquetas de Triaje (Triage Labels)

Las habilidades operan bajo cinco roles canónicos de triaje. Este archivo mapea dichos roles con las cadenas de texto (etiquetas) reales que utiliza el rastreador de problemas de este repositorio.

| Rol en mattpocock/skills | Etiqueta en nuestro rastreador | Significado |
| :--- | :--- | :--- |
| `needs-triage` | `needs-triage` | El mantenedor debe evaluar este problema |
| `needs-info` | `needs-info` | Esperando más información por parte del reportero |
| `ready-for-agent` | `ready-for-agent` | Totalmente especificado, listo para un agente (AFK) |
| `ready-for-human` | `ready-for-human` | Requiere implementación por parte de un humano |
| `wontfix` | `wontfix` | No se tomarán medidas al respecto |

## Instrucciones de uso

Cuando una habilidad mencione un rol (por ejemplo, "aplica la etiqueta de triaje listo para agente"), utiliza la cadena de texto correspondiente de la columna derecha de esta tabla.

**Personalización**: Edita la columna de la derecha ("Etiqueta en nuestro rastreador") para que coincida con el vocabulario que utilices actualmente en tu proyecto.
